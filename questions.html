Frequently Asked Questions

What are the different ways to select an element in the DOM?

DOM-এ element সিলেক্ট করার বিভিন্ন উপায় তার মধ্যে কয়েকটি হলো:- ID দিয়ে একটি এলিমেন্ট সিলেক্ট করা যায়, ক্লাসের নাম দিয়ে সিলেক্ট করা যায়, tag নাম দিয়ে সিলেক্ট করা যায় ইত্যাদি |

What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, আর textContent এর মধ্যে পার্থক্য হলো:-
1. innerHTML: টেক্সট সহ HTML ট্যাগও দেখায় বা পরিবর্তন করে।
2. innerText: শুধু টেক্সট দেখায়, CSS অনুসারে দেখা যায় এমন টেক্সটই দেখাবে।
3. textContent: সব টেক্সট দেখায়, এমনকি হাইড করা থাকলেও।

What is event delegation in the DOM?

Event delegation মানে হচ্ছে parent element-এর মাধ্যমে child element গুলোর event হ্যান্ডেল করা।

What is event bubbling in the DOM?

Event bubbling হচ্ছে যখন কোনো child element-এ event ঘটে, তখন সেই event টা উপরের দিকে চলে যায়।

How do you create, add, and remove elements using JavaScript?

JavaScript দিয়ে কিভাবে নতুন এলিমেন্ট তৈরি, যোগ ও মুছতে হয় তাহলে:-
1. তৈরি করা: const p = document.createElement("p");
p.textContent = "Hello";
2. যোগ করা: document.body.appendChild(p);
3. মুছে ফেলা: p.remove();
